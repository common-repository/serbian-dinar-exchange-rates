"Serbian Dinar Exchange Rates" gives the users from Serbia currency exchange rate widget.

== Description ==

"Serbian Dinar Exchange Rates" is currency exchange rate plugin that gives the users from Serbia a possibility to see changes in rates of foreign currencies.
"Serbian Dinar Exchange Rates" use free webamster tool from online service called [Kursna Lista](http://www.kursna-lista.info/kursna-lista-na-vasem-sajtu)


Main features of Widget "Serbian Dinar Exchange Rates":

* User friendly and easy to use.
* Tested with other popular plugins.
* Choose a format/look of your currency exchange rate.
* Show or hide percentage in rate changes.
* Show changes in time.
* Show the date.
* Adjusting width and hight.

== Installation ==

### Here is the instruction:

1. First download Serbian Dinar Exchange Rates.
2. Upload the "Serbian Dinar Exchange Rates" from your admin panel.
3. Then activate the plugin. 
4. Then go to your widget page and add "Serbian Dinar Exchange Rates" to your sidebar.

== Frequently Asked Questions ==


### How to use Serbian Dinar Exchange Rates

First of all log into your admin panel, then go to your "Appearance" page from your left admin menu, then go to your "Widgets" page, then drop and down "Serbian Dinar Exchange Rates" widget to your sidebar and configure the like box.
